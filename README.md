# صندوق عائلة علوان - Alwan Family Fund

تطبيق ويب احترافي ومتجاوب لإدارة صندوق العائلة بشكل كامل، مع تصميم عصري وأنيق يشبه الأنظمة البنكية.

## المميزات الرئيسية

### 1. الهوية والتصميم
- واجهة عربية بالكامل (RTL)
- تصميم احترافي بألوان:
  - الأزرق الداكن (#0F172A)
  - الذهبي (#D4AF37)
  - الأبيض (#FFFFFF)
- دعم الوضع الفاتح والداكن
- تصميم متجاوب (Responsive Design)
- رسوم بيانية وأيقونات احترافية

### 2. إدارة الأعضاء
- إضافة وتعديل وحذف الأعضاء
- بيانات العضو (الصورة، رقم الهوية، الجوال، تاريخ الانضمام)
- حالات العضوية (نشطة، مجمدة، موقوفة)
- بحث وتصفية متقدمة

### 3. إدارة الاشتراكات
- تسجيل الاشتراكات الشهرية
- متابعة المسددين وغير المسددين
- إشعارات تلقائية للمتأخرين
- تصدير البيانات (Excel, PDF)

### 4. إدارة القروض
- تقديم طلبات قرض
- حساب الأقساط تلقائياً
- جدول سداد شامل
- حالات القرض (قيد المراجعة، معتمد، مرفوض، مكتمل)

### 5. إدارة المصروفات
- إضافة وتصنيف المصروفات
- إرفاق الفواتير
- تقارير شهرية وسنوية

### 6. التقارير والإحصائيات
- تقارير مالية شاملة
- رسوم بيانية تفاعلية
- تصدير (PDF, Excel)

### 7. نظام الصلاحيات
- **مدير النظام:** إدارة كاملة
- **أمين الصندوق:** إدارة العمليات المالية
- **عضو الصندوق:** عرض بيانات شخصية فقط

## التقنيات المستخدمة

- **Frontend:** HTML5 + CSS3 + JavaScript (ES6+)
- **Backend:** Firebase (Authentication + Firestore + Storage)
- **Charts:** Chart.js
- **Design:** Glassmorphism + Smooth Animations
- **PWA:** قابل للتحويل إلى تطبيق موبايل

## البدء السريع

1. استنسخ المستودع
2. قم بتحديث بيانات Firebase في `js/config.js`
3. افتح `index.html` في المتصفح
4. سجل الدخول بحسابك
